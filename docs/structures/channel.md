Channel Class
==============
>`self.name`
> 
> Type: `String`
> 
> Channel Name

>`self._id`
> 
> Type: `String`
> 
> Channel's ID

>`self.members`
> 
> Type: `Array`
> 
> A list of all the members that have access to a channel
> 
> Note: Disabled on larger servers

>`self.messages`
> 
> Type: `Array`
> 
> A list of all messages in the channel
> 
> [[Message Class](message.md)]
